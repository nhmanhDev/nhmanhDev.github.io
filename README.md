# Nguyen Hung Manh - AI Portfolio

This is my personal portfolio, built to showcase my AI engineering projects, skills, and experience.

## ✨ Features

- **Modern & Dark UI**: Sleek, professional design tailored for an AI developer.
- **Experience Timeline**: Vertical timeline component highlighting professional and academic journey.
- **Projects Showcase**: Interactive cards demonstrating RAG Chatbots, Computer Vision scoring systems, and NLP applications.
- **Fully Responsive**: Optimized for desktop and mobile devices.

## 🛠 Tech Stack

- **Frontend**: React.js, React Bootstrap, CSS3
- **Animations**: react-tsparticles, react-parallax-tilt, react-vertical-timeline-component
- **Routing**: react-router-dom

## 🚀 Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm start
```

## 🌐 Deployment

This project is configured to automatically deploy to GitHub Pages.
```bash
npm run deploy
```
